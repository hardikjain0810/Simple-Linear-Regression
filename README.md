# Simple-Linear-Regression
<h2><b>Project Overview</b></h2>
<p>This project demonstrates the complete implementation of Simple Linear Regression using python and scikit-learn.</p>
<p>The goal is to understand how a single independent variable influences a dependent variable and evaluate the model performance using statistical metrics.</p>
<h2>Tech stack and tools</h2>
<ul>
  <li>Python</li>
  <li>Numpy</li>
  <li>Pandas</li>
  <li>Matplotlib</li>
  <li>Seaborn</li>
  <li>Scikit-learn</li>
</ul>
